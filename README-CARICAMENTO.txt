PACCHETTO SEO VELIXTOR VPN

- guide/ (pagine hub SEO)

COME CARICARE SU GITHUB

1. Fai backup del repository.
2. Carica la cartella seo-assets nella root del repo.
3. Carica la cartella guide nella root del repo.
4. Carica la cartella faq nella root del repo.
5. Sostituisci robots.txt e sitemap.xml.
6. Apri index.html del tuo sito e inserisci il contenuto di homepage-snippet.html.
7. Commit e attendi deploy GitHub Pages.

NOTE

- Questo pacchetto non tocca pagamenti o backend.
- Le guide linkano al sito principale Velixtor.
- La homepage snippet è facoltativa ma consigliata.
