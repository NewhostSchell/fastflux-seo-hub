- seo-assets/seo.css
- guide/ (22 pagine iniziali + hub index)

COME CARICARE SU GITHUB
1. Fai backup del repository fastflux-site.
2. Carica la cartella seo-assets nella root del repo.
3. Carica la cartella guide nella root del repo.
4. Carica la cartella faq nella root del repo.
5. Sostituisci robots.txt e sitemap.xml con quelli di questo pacchetto.
6. Apri index.html del tuo sito e inserisci il contenuto di homepage-snippet.html nella homepage, idealmente sotto la sezione principale o prima del footer.
7. Commit e attendi il deploy GitHub Pages.

NOTE
- Questo pacchetto non tocca pagamenti, supporto, Stripe, PayPal o API FastFlux.
- Le guide linkano alle tue pagine esistenti: /installazione.html /pagamenti.html /supporto.html
- La homepage snippet è facoltativa ma consigliata.
